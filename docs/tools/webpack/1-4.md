# webpack配置

webpack.config.js 是运行在node环境的, 所以必须遵循CommonJs规范

```js
const path = require('path')

module.exports = {
    mode: 'development',
    entry: './src/main.js',
    output: {
        filename: 'bundle.js',
        path: path.join(__dirname, 'dist')
    },
    module: {
        rules: []
    }
}
```
